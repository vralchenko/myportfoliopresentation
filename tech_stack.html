Presentation Technology Stack

Core & UI

  • React + TypeScript: Main framework for building a scalable and type-safe frontend architecture.
  • Tailwind CSS: Used for layout and styling (Glassmorphism effects).
  • Lucide React: Vector icons for navigation and UI elements.

Animation & Visual Effects

  • Framer Motion: Core engine for slide transitions, data fly-in animations, and interactive hover states.
  • AnimatePresence: Used for managing entry and exit animations of components.

Audio Engine

  • Custom Music Manager: React-based implementation using hooks (useRef/useEffect) for cross-slide audio persistence.
  • Dynamic Fading: Programmatic volume control for smooth Fade-In and Fade-Out transitions.
  • SoundHelix API: Used for audio streaming of ambient tracks.

Interactive Simulations

  • PostMessage API: Cross-origin communication between the presentation deck and embedded demo apps.
  • Simulation Generators: Custom JS logic for real-time automated data entry and interaction demos.

Localization & Infrastructure

  • Localization Engine: Custom context-based system supporting 4 languages (EN, DE, UA, RU).
  • Vite: Development server and build tool for high-performance delivery.
